🎮 GAME MECHANICS FRAMEWORK

   Vehicles (3 Types)
   ├── 🚙 Jeep        (1500 kg, balanced)
   ├── 🏍️ Bike        (200 kg, fast)
   └── 🚚 Truck       (3000 kg, powerful)

   Upgrades (4 Categories)
   ├── ⚙️  Engine      (5 levels, +torque/speed)
   ├── 🔧 Suspension  (5 levels, -bounce/+stable)
   ├── 🛞  Tires       (5 levels, +friction)
   └── 🏎️  4WD         (3 levels, +traction)

   Levels (15 Total)
   ├── 🟢 Easy        (Levels 1-2, tutorial)
   ├── 🟡 Medium      (Levels 3-5, challenge)
   ├── 🟠 Hard        (Levels 6-8, demanding)
   └── 🔴 Extreme     (Levels 9-15, maximum)

   Terrain Types (4)
   ├── 🏔️  Smooth Hill (standard racing)
   ├── 🪨 Rocky       (bumpy terrain)
   ├── ❄️  Ice         (low friction)
   └── 💧 Water       (flooded areas)

🚀 QUICK START

   1. Virtual Environment:
      $ python -m venv venv
      $ venv\Scripts\activate

   2. Install Dependencies:
      $ pip install -r requirements.txt

   3. Initialize Database:
      $ python -c "from backend.database import init_db, seed_db; \
                   init_db(); seed_db()"

   4. Run Backend Server:
      $ python backend/main.py

   5. Access API:
      → http://localhost:8000
      → http://localhost:8000/docs (Swagger UI)
